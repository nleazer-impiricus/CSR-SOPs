SOP — Detailed Steps

Search the Zendesk Knowledge Base

Open Zendesk and search for the drug name. Use both brand and generic names if you know them. Look for existing articles or processes.

🔀Did you find an existing KB article for this drug?
Yes → Follow the documented process and reply to the HCP in their Zendesk ticket. You're done!
No → Continue to Step 2.

Search #concierge-call-center in Slack

Use Slack search within the #concierge-call-center channel. Search the drug name — someone may have already researched this. Check for Coordinator-approved answers.

🔀Did you find a previously approved process in Slack?
Yes → Follow that process and reply to the HCP in Zendesk.
📌
KB Gap Flag: If you found the answer in Slack but it's not in the Zendesk Knowledge Base, flag this to the Coordinators so they can add it to the KB. Post a quick note in #concierge-call-center: "Used Slack-approved process for [Drug Name] — not currently in KB. Flagging for KB addition."
No → Continue to Step 3.

Run the Perplexity AI Prompt

Open Perplexity and use the saved prompt template. Type /samples [drug name] to run it. Perplexity will search multiple sources and return a structured answer with links.

⚠️
You must verify everything Perplexity returns. AI can hallucinate links, phone numbers, and processes. Every link must be clicked. Every phone number must be confirmed on the manufacturer's website. Never send unverified AI output to an HCP.

🛡️ Verification Checklist — Complete Before Proceeding

  • U.S. manufacturer site: Confirm the website is the U.S. version of the manufacturer site (look for .com with U.S. content, not .co.uk, .ca, .eu, etc.).
  • Forms accept U.S.: If there's a request form, verify that "United States" is available as a country option and that the form accepts U.S. NPI numbers, zip codes, and state selections.
  • Every link works: Click every URL Perplexity returned. Does it load? Does it go where Perplexity said it does?
  • Phone numbers verified: Find each phone number on the manufacturer's own website — not just in the AI output.
  • Program exists: Confirm the sample/rep/demo program is actually described on the official site, not just inferred by the AI.
  • Information is current: Check for copyright dates, last-updated notes, or any indication the page is outdated or discontinued.
🔀Did Perplexity return useful results that passed the verification checklist?
Yes → Skip to Step 5 (Coordinator Approval).
No / Couldn't verify → Continue to Step 4.

Manual Google Search — HCP Site Focus

Search Google to find the official manufacturer website for this drug. Focus on the Healthcare Professional (HCP) version of the site — not the patient-facing site.

🔍
How to find the HCP site:
Look for URLs containing /hcp, /professional, /medical, or subdomains like hcp.drugname.com. Look for a link or toggle that says "For Healthcare Professionals" on the patient site. See the Google Tips tab for search shortcuts.

Once on the HCP site, look for any of these:

  • Sample request forms or portals
  • "Request a Rep" or "Contact a Representative" pages
  • Phone numbers for HCP/medical inquiries
  • Demo device or trainer request forms
  • Patient start / bridge program info
🇺🇸
U.S. Site Check: Before recording any info, confirm you are on the U.S. version of the manufacturer site.
🔀Did you find official U.S. manufacturer information?
Yes → Run through the Verification Checklist (see Step 3), then continue to Step 5.
No → Post to #concierge-call-center and flag it for help from the team. Include what you've already tried.

Post to #concierge-call-center for Coordinator Approval

Since this is a new process (not already in the KB), you must get Coordinator approval before sending anything to the HCP. Post your findings using the format in the Posting Format tab.

⏳
Do not reply to the HCP until a Coordinator approves your findings. If you're unsure about anything, say so in your post — the Coordinator or another team member can help.
🔀Coordinator response?
Approved → Reply to the HCP in their Zendesk ticket with the approved information.
Revisions requested → Make changes per Coordinator feedback and resubmit in Slack.

Perplexity AI — What It Is & How to Use It

What is Perplexity?

Perplexity is an AI-powered search engine. Unlike ChatGPT (which generates answers from memory), Perplexity actually searches the internet in real-time and shows you the sources it found. Think of it like Google, but instead of giving you a list of blue links, it reads the pages for you and writes a summary with citations.

Why are we using it?

For our use case — finding manufacturer sample programs, rep contacts, and HCP resources — Perplexity is helpful because it can search multiple sites at once and pull together information that would take several Google searches to find manually. The prompt template we've built tells Perplexity exactly what to look for and how to structure its answer.

The Critical Rule: Always Verify

Perplexity (and all AI tools) can and will make mistakes. It may return links that don't exist, phone numbers that are wrong, or confidently describe programs that aren't real. You are responsible for verifying every piece of information before it goes to an HCP.

🚫
Never copy-paste Perplexity's output directly to an HCP. Always click every link, confirm every phone number on the manufacturer's actual website, and verify the process exists.

Getting Started (One-Time Setup)

  1. Go to perplexity.ai and create a free account.
  2. Once signed in, you'll see the search bar — this is where you'll run prompts.
  3. Save the prompt from the Prompt Template tab as a Perplexity template so you can reuse it quickly.
  4. To save a template: click the "Library" icon in the left sidebar → "Prompts" → "Create Prompt" → paste the prompt → set a shortcut like /samples.
  5. Now anytime you need it, just type /samples in the search bar and fill in the drug name.

Running a Search

  1. Open Perplexity and click the search bar.
  2. Type /samples — your saved template will appear. Select it.
  3. Replace [drug name] with the medication you're researching.
  4. Press Enter. Perplexity will search the web and return a structured answer.
  5. Review the answer — check each cited source by clicking the numbered references.
  6. Verify every link and claim using the Verification Checklist in the SOP Steps tab before using anything from the results.
💡
Tip: If Perplexity's first answer is incomplete, you can ask follow-up questions in the same thread — for example: "Can you also check for a demo trainer or device request form?" or "What is the HCP phone number for [manufacturer]?"

How to Identify the HCP Version of a Site

Most manufacturer drug websites have two versions — one for patients and one for healthcare professionals. You always want the HCP version. Here's how to spot it:

  • URL clues: Look for /hcp, /pro, /professional, /medical, or /physician in the URL. Some use subdomains like hcp.drugname.com.
  • Site toggle: Many patient sites have a small link or button in the header that says "For Healthcare Professionals" or "HCP Site" — click that.
  • Confirmation gate: HCP sites often ask you to confirm you're a licensed healthcare professional before entering. That's a good sign you're in the right place.

🇺🇸 Confirming It's a U.S. Site

Many manufacturers have separate regional websites. Before using any information, confirm you're on the U.S. version:

  • Check the URL: Avoid domains ending in .co.uk, .ca, .eu, .com.au, etc.
  • Check the footer: Look for "United States" in the country selector or footer text.
  • Check the header: Some sites have a country/region selector at the top — make sure it's set to the U.S.
  • Check forms: If there's a request form, verify it accepts U.S. NPI numbers, U.S. state selections, and U.S. zip codes.

What to Look For on the HCP Site

Once you're on the HCP site, navigate to these areas:

  • Resources / Support section: Sample request forms, starter kits, bridge programs
  • Contact / Request a Rep: Forms to request a sales representative visit
  • Getting Started / Getting Patients Started: Often contains sample info, onboarding materials
  • Footer links: Phone numbers for medical information, often an 800 number
✅
Trust only official manufacturer domains. If you find info on a third-party site, forum, or aggregator, go to the manufacturer's actual website to confirm it. Only official sources are acceptable.
