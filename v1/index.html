HCP Pharma Resource Lookup

Use this workflow whenever an HCP writes in requesting drug samples, rep contacts, or official pharmaceutical resources. Follow each step in order — do not skip ahead.

📨
Trigger An HCP contacts us requesting drug samples, a pharmaceutical rep, or official drug resources.
01 Check the Internal Knowledge Base First Stop

Before anything else, search our internal KB. We have pre-verified processes for many common drugs — use these first, as they are confirmed correct.

Search the KB by drug name (brand name and generic)
Check if there's an existing process, verified link, or rep contact on file
If found → verify the link is still live, then proceed to Step 5 (Send)

02 Use the Custom Perplexity AI Prompt AI-Assisted

Open Perplexity and use the saved template prompt below. This prompt is specifically designed to surface official pharmaceutical HCP resources. Copy it, fill in the drug name, and run it.

📋 Saved Template Prompt — Copy & Use in Perplexity
[PASTE YOUR CUSTOM PERPLEXITY PROMPT HERE] Tips for using this template: · Replace [DRUG NAME] with the brand or generic name · Save this as a Collection in Perplexity for quick access · Do not edit the core instructions — only fill in the drug name
Open Perplexity → navigate to your saved prompt template
Enter the drug name (try both brand and generic if needed)
Run the prompt and review the results
03 Verify the AI Results Required

AI can be wrong. Every result from Perplexity must be manually verified before you use it. Do not send any link or contact directly from AI output.

Open each link Perplexity provides — confirm it loads
Confirm the site is the official manufacturer's domain (e.g., drugname.com, manufacturer.com)
Confirm the page is the HCP/healthcare professional version of the site (look for an HCP portal or professional section)
Confirm samples or rep-contact info is present and current
⚠️
Red flags to watch for: Patient-facing pages (not HCP), third-party or aggregator sites, links that redirect unexpectedly, outdated or discontinued drug pages, or anything that looks unofficial.

04 Manual Google Search for Official Site Fallback

If Perplexity couldn't give you a verified result, search Google manually. The goal is to find the drug's official manufacturer website and locate its HCP portal.

Search: "[Drug Name] HCP" or "[Drug Name] healthcare professional"
Look for the official manufacturer domain — avoid third-party sites (e.g., Drugs.com, GoodRx, Wikipedia)
Navigate to the HCP or Professional section of the site
Locate samples, rep contact, or ordering info for HCPs
Verify the URL — it should be the manufacturer's own domain, not a redirect
Step 05 · Final Step

Verify the Link & Send to HCP

Before sending anything — whether it came from the KB, Perplexity, or manual search — run through this final verification checklist every single time.

Click the link yourself — confirm it loads and is not broken
Confirm it is the official manufacturer domain (not a third-party)
Confirm it is the HCP-facing version of the resource
Confirm the resource is relevant to what the HCP asked for (samples, rep, ordering)
Send to HCP with a clear, brief note explaining what the link provides
If verified and useful, add the drug + resource to the internal KB for future use
